这也太厉害了吧！screenshot-to-code 是一款使用 AI 将屏幕截图、模型和 Figma 设计转换为干净、实用的代码的简单工具。现在支持 Claude Sonnet 3.5 和 GPT-4o！
&lt;!--more--&gt;

支持的堆栈：

- HTML &#43; Tailwind
- HTML &#43; CSS
- React &#43; Tailwind
- Vue &#43; Tailwind
- 引导
- Ionic &#43; Tailwind
- SVG

支持的 AI 模型：

- Claude Sonnet 3.5-最佳模型！
- GPT-4o——也推荐！
- DALL-E 3 或 Flux Schnell（使用 Replicate）用于图像生成
