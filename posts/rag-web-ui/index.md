# RAG Web UI - 基于RAG的智能对话系统，帮助你用自己的知识库快速搭建智能问答

支持PDF、DOCX、Markdown、Text等多种文档格式 文档自动分块和向量化，支持异步文档、增量处理 支持上下文多轮对话，回复中标注信息来源 前后端分离设计，分布式文件存储 支持云端和本地模型部署
&lt;!--more--&gt;

一个基于RAG的智能对话系统：RAG Web UI ，它可以构建基于自有知识库的智能问答系统，提供准确可靠的知识问答服务

支持PDF、DOCX、Markdown、Text等多种文档格式

文档自动分块和向量化，支持异步文档、增量处理

支持上下文多轮对话，回复中标注信息来源 前后端分离设计，分布式文件存储

支持云端和本地模型部署

github：[https://github.com/rag-web-ui/rag-web-ui/tree/main](https://github.com/rag-web-ui/rag-web-ui/tree/main)
